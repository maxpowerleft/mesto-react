# Сервис Mesto (React version)

Проектная работа с поддержкой от команды **Яндекс.Практикум**.

Данный проект "Mesto" представляет из себя раннее написанный фото-сервис (JavaScript ES6). 
Более подробно с проектном можно ознакомиться по ссылке: 
### [Открыть репозиторий с проектом](https://github.com/maxpowerleft/mesto)

## Особенности React-версии:
* Перенесена разметка проекта
* Функциональные части описаны в виде компонентов
* Используется создание и подключение контекста

## Планируется реализовать:
* Валидация форм
* UX - доработки

## Используемые технологии:
* HTML5
* CSS3
* Адаптивная верстка (flex, grid)
* JavaScript ES6
* React
* Методология БЭМ (в структуре и разметке проекта)
* API
* Webpack
* Babel
